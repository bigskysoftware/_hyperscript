6. The exact bug report pattern

This was the failing case: all four events targeting window, including resize from window which the pre-fix code tried to hand to ResizeObserver.observe(window). After the fix, window-targeted resize falls through to the native resize event. (The htmx:afterSettle spec is parsed and installed even though htmx isn't loaded -- it just never fires.)
