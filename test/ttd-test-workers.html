TTD Worker Race

Two workers each pick a random delay (200–1000ms) before returning. Click Race to dispatch both — whichever finishes first appears first in the log. Then use the console to time-travel through the results.

Serve from project root: npx serve .
Open with: /test/ttd-test-workers.html?_ttd

Ready

    Try in the console

    ttd.steps()          // see the interleaved timeline
ttd.streams          // see all execution streams
ttd.back()           // undo last DOM change
ttd.back()           // undo the one before that
ttd.forward()        // redo
ttd.forward()        // redo
ttd.resume()         // return to live execution

// run the race again, then:
ttd.find("Alpha")    // find all Alpha-related steps
ttd.find("Beta")     // find all Beta-related steps
