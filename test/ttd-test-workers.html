TTD Worker Race

Two workers each pick a random delay (200–1000ms) before returning. Click Race to dispatch both — whichever finishes first appears first in the log. Then use the console to time-travel through the results.

Serve from project root: npx serve .
Open with: /test/ttd-test-workers.html

Ready

    Inspect the run

    Open the debugger with Ctrl+.. The Timeline pane shows the interleaved steps from both workers; click any row to step through the recorded DOM changes.

    The same data is available headlessly at _hyperscript.debugger.ttd:

    var ttd = _hyperscript.debugger.ttd;
ttd.steps()       // the interleaved timeline
ttd.goto(n)       // time-travel to step n
ttd.resume()      // return to live execution
ttd.clear()       // wipe recorded history
