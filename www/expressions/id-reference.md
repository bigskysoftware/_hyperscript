## The `id reference` Expression

An id reference uses CSS selector syntax to reference a single DOM element by its `id` attribute.

### Examples

```html
<div _="on click add .clicked to #another-div">
  Add the .clicked class to another div!
</div>
```

### Syntax

```ebnf
#<id-value>
```
