_hyperscript

A scripting language for the web

Enhance HTML with concise DOM, event and async features. Make writing interactive HTML a joy.

example.html
on pointerdown
  repeat until event pointerup from the document
    set rand to Math.random() * 360
    transition
      *background-color
      to `hsl($rand 100% 90%)`
      over 250ms
  end
  transition *background-color to initial
Install <script src="https://unpkg.com/hyperscript.org@0.9.14"></script> copy
Events
<button _="on click
              send hello to <form />">
  Send
</button>

<form _="on hello
           alert('got event')">

Easily send and receive events. Filter, queue or debounce them.

Async
<div _="on click
          wait 5s
          send hello to .target">

<div _="init
          fetch /api as json
          put result into me">

No promises, async/await or callbacks. _hyperscript handles async transparently.

DOM
<div _="on click
          tell <p/> in me
            add .highlight">

<div _="on click
          take .active from .tabs
          for me">

CSS selectors and positional operators make DOM access a breeze.

Reactivity
<input type="text" id="name" />
<h1 _="when #name.value
          changes
        put it into me"></h1>

Watch any value and react when it changes. Bind values for two-way sync.

Enhance existing code

_hyperscript works side by side with existing JavaScript. You choose what and when to enhance. Use locality of behaviour, or external behaviors.

JS Interop
<div _="init js
          alert('Hello from JS!')
        end">

<div _="init js(haystack)
          return /needle/gi
            .exec(haystack)
        end">
Debugging & Extending
on click breakpoint

Graphically debug and step through code as it runs.

_hyperscript is natively extensible — add new commands or expressions using vanilla JS.

Remember HyperCard?

An xTalk syntax inspired by HyperTalk and AppleScript, running natively in your browser.

Behaviors
<div _="install Draggable(dragHandle: .titlebar)">

Reusable behaviors that can be installed on any element.
